# Anton Experiment 

test workflow

Interactive Three.js scene featuring a floating obsidian monolith surrounded by orbiting shards.

## Live Demo

https://bleiglanz.github.io/anton-experiment

## Interactions

- **Click a shard** — bursts it outward, pushing nearby shards
- **Click the monolith** — attracts all shards back to center
- **Double-click** — resets all shards to their orbits
- **Hover** — amber glow on objects
- **Drag** — orbit camera
- **Scroll** — zoom
- **Right-drag** — pan
